# CppUnitTest

### Build Instructions
Before editing any code, or attempting to run, do these commands
1. `mkdir out out/build out/arc out/obj`
2. `cmake .`
3. `make`

**For testing, run the example test**
- `./out/build/Main tests/ExampleQuiz.txt`
